# Personal-Finance-ChatBot-Gen-AI-
The Personal Finance Chatbot helps users plan budgets, track expenses, and improve savings using an easy form-based interface. It supports Indian Rupees (₹), shows insights with charts, and provides friendly advice using IBM Granite, while Groq delivers accurate spending analytics.


Pre-requisites:
1.	Python Programming – https://docs.python.org/3/
2.	FastAPI Integration - https://fastapi.tiangolo.com/
3.	Groq LLM API (for analytics) – https://console.groq.com/docs
4.	IBM Granite via Ollama (for chatbot) – https://ollama.com/library/granite
5.	 Ollama Runtime Setup – https://ollama.com/download
6.	Vanilla HTML/CSS/JavaScript – https://developer.mozilla.org/en-US/docs/Web
7.	Chart.js for Data Visualization (CDN) – https://www.chartjs.org/docs/latest/
8.	Pydantic for Data Validation – https://docs.pydantic.dev/latest/
9.	Uvicorn ASGI Server – https://www.uvicorn.org/
10.	Environment Configuration (.env) – https://pypi.org/project/python-dotenv/
